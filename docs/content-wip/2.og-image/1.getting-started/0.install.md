Using [Nuxt SEO]? You already have this module installed :)

## Install

```bash
npm install -D nuxt-og-image
#
yarn add --dev nuxt-og-image
#
pnpm add -D nuxt-og-image
```

## Setup

Add the module to your `nuxt.config.ts`.

```ts
export default defineNuxtConfig({
  modules: [
    'nuxt-og-image',
  ],
})
```
